#### 代理工具：
> 配置适用于基于 Clash Meta 内核的代理工具，如：
> - **[Mihomo Party](https://github.com/pompurin404/mihomo-party/releases):** 大部分功能GUI化，支持 Sub-Store ，脚本覆写（YAML JavaScript），策略组图标 及 策略组隐藏。
