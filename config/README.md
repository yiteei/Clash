### 配置介绍
这份配置文件为本人使用而设计的 [Clash Meta](https://github.com/MetaCubeX/mihomo) 核心配置，旨在满足本人日常使用需求。

#### 主要功能：
> - **分流管理:** 确保必要的流量直连，其他流量通过代理处理。
> - **节点管理:** 支持多个代理集合，切换节点以保证稳定性。
> - **隐私保护:** 基本 DNS 配置，防止 DNS 污染。
> - **广告拦截:** 简单的广告过滤规则，提升浏览体验。
